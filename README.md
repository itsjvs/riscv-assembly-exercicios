# Exercícios de Assembly RISC-V

Coleção de 9 exercícios em Assembly RISC-V focados nos fundamentos de programação de baixo nível.

## Tópicos abordados:
- Manipulação de registradores
- Operações aritméticas
- Desvios e laços (branching e loops)
- Acesso à memória

## Estrutura:
Cada exercício contém:
- Enunciado do problema
- Solução em Assembly

## Como executar:
Utilize ferramentas como RARS ou Venus para montar e executar os arquivos `.asm`.
